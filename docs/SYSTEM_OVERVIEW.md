## 医疗设备登记管理系统 · 系统概述

### 一、系统定位

本系统全面替代传统医疗设备使用、保养纸质登记本，依托企业微信端与 PC 端双入口搭建电子化管理体系，为医工部、技师及临床护士提供设备使用、维护、消毒全流程的电子化登记与管理能力，解决纸质登记本易丢失、统计效率低、信息追溯难的行业痛点，助力医院实现设备管理的规范化、精细化升级。

### 二、核心用户群体及职责

- **医工部**：负责设备全生命周期管理、维护记录审核、设备状态监控与系统权限配置；
- **技师**：执行设备日常维护、故障报修、校准质控等操作，并完成相关信息登记；
- **临床护士**：高频使用设备时，完成设备使用登记、消毒记录填报及设备状态确认。

### 三、系统访问入口及功能适配

- **企业微信端**：适配护士、技师移动端操作场景，支持扫码登记、快速提交设备使用 / 维护记录，满足临床一线高效操作需求；
- **PC 端**：面向医工部及管理人员，提供完整的设备管理、数据多维度查询、报表一键导出等全功能操作能力，支撑设备管理后台运营与数据分析。
