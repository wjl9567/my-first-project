PDF 导出中文字体说明
====================

若导出的 PDF 出现中文乱码，可在此目录放置中文字体文件（.ttf）供程序使用。

推荐文件名（任选其一即可）：
  - SimSun.ttf   （宋体）
  - SimHei.ttf   （黑体）
  - msyh.ttf     （微软雅黑）
  - CJK.ttf      （任意支持中文的 TTF）

Windows 用户可从以下目录复制字体（注意需为 .ttf，.ttc 需本机 ReportLab 支持或自行转换）：
  C:\Windows\Fonts\

程序会优先使用本目录下的字体，其次尝试系统字体（含 Windows Fonts 下的 simsun.ttc、msyh.ttc 等）。
